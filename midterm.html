HTML이란?

HTML은 하이퍼텍스트 마크업 언어(Hypertext Markup Language)의 약자로, 웹페이지를 만들 때 사용되는 문서 형식입니다.

하이퍼텍스트(Hypertext)란 하이퍼링크 (Hyperlink)를 담고 있는 문서를 뜻합니다.하이퍼링크는 문서들이 서로 연결되어 있는 고리를 말합니다.

CSS란?

CSS는 Cascading Stylesheet의 약자이며 HTML 같은 마크업 언어로 작성된 문서를 꾸미기 위해 사용합니다. HTML문서의 head요소 안에 style 요소를 널고 CSS코드를 적으면 해당문서를 다양하게 꾸밀 수 있습니다.

CSS에 대한 더 자세한 설명은 CSS문서를 참고하세요.

이미지 넣기

이미지를 넣으려면 img 요소를 사용합니다.

src 속성에는 이미지의 경로(URL)를 적습니다.

이미지 요소에는 항상 alt 속성을 사용하여 대체 텍스트를 명시해주어야 합니다."alt"는 "alternative text"dml 약자로, 이미지를 표시 할 수 없을 때 이미지를 대신하여 나타낼 텍스트를 뜻합니다.

블럭 요소

다음은 HTML에서 자주 사용되는 요소들입니다.

  1. P:단락(Paragraph)
  2. h1:큰제목(Heading 1)
  3. ul:순서 없는 목록(unorderedList)
  4. ol:순서 있는 목록(Ordered List)
  5. table:표(table)
